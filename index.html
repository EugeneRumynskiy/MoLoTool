MoLoTool
Transcription Factor Motif Location Toolbox (MoLoTool) is an interactive user-friendly web application which allows user to scan one's sequences for a presence of Transcription Factor Binding Sites (TFBS). Current binding models consist of Homo Sapiens Comprehensive Model Collection (HOCOMOCO).
HOCOMOCO
HOmo sapiens COmprehensive MOdel COllection (HOCOMOCO) contains Transcription Factor (TF) binding models represented as classic Position Weight Matrices (PWMs, also known as Position-Specific Scoring Matrices, PSSMs) and precalculated score thresholds (see HOCOMOCO-v10 paper for details).
Manual
To show a basic example click "Demo" button at the top right corner of the window.

In order to locate Transcription Factor Binding Sites follow these steps:
  1. Add desired binding models from HOCOMOCO. To do that use search by name or family.
  2. Then upload sequence file or type sequences manually. FASTA format is supported.
  3. Adjust P-value.
  4. If sequences are long enough (100bp+), please wait a moment. We are still working on performance improvements.
  5. Detailed information about binding sites that was found, provided with the table at the bottom of the page.
